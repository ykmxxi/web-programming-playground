제목 태그들과 문단 태그

용도에 적합한 태그 사용하기

페이지나 섹션, 주요 오소의 제목은 h1 ~ h6 태그를 사용한다.
숫자가 높을 수록 낮은 단계의 제목이 된다.

본문은 문단별로 p 태그로 감싼다. p 태그끼리는 자동으로 줄바꿈이 된다.
문단에 관계없이 이렇게


원하는 횟수로 줄바꿈을 하기 위해서는 br 태그를 사용하면 된다.
br 태그는 열고 닫을 필요가 없는 몇 안 되는 태그 중 하나인데 남용하지 않는 것이 좋다.

위와 같이 hr 태그를 사용하면 가로줄을 표시할 수도 있다.

이   렇   게 글자 사이의 공백을 여럿 두로면 & n b s p ; 를 붙여서 입력해야 한다

마지막으로, 여기서는 보이지 않지만 페이지 원본 보기에서 주석을 확인할 수 있다
