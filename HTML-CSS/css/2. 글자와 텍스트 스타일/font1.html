font-size 속성은 말 그대로 글자의 크기를 지정합니다.
단위로는 px과 %, em, rem이 사용되죠.

px는 절대값으로서 픽셀 단위입니다.
100%는 1em으로, 이들은 부모 요소와의 상대적 크기를 나타내죠.
rem은 html 요소와의 상대적 크기를 가지므로, 요소의 중첩에 영향을 받지 않습니다.
pt는 1인치/72로, 프린트할 컨텐츠에 사용됩니다.
